🌐 Networking Basics

Learn how your browser connects to the internet and fetches web pages.

🔗 Key Concepts

  • IP Address: Your device’s unique ID
  • DNS: Translates domain names to IPs
  • HTTP/HTTPS: Protocols for communication

🧪 Example

When you visit bloombyte.com, your browser:

  1. Asks DNS for the IP
  2. Connects via HTTP
  3. Downloads HTML, CSS, JS
